Suffix automaton

In computer science, a suffix automaton is a data structure that efficiently represents the suffixes of a string. For example, a suffix automaton for the string "suffix" can be queried for other strings; it will report "true" for any of the strings "suffix", "uffix", "ffix", "fix", "ix" and "x", and "false" for any other string.

A suffix automaton is a type of finite state machine. It can be regarded as a compressed suffix trie.
